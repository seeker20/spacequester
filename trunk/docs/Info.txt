Es gibt drei Ordner im Ordner trunk

-trunk
	|-docs
	|-src
	|-sql
	|-messages
	
im Ordner docs befinden sich allgemein Dokumente zum Projekt.

im Ordner src befindet sich er aktuelle Quelcode zum Projekt. 
	Er Steht komplet unter der GPL

im Ordner sql gibt es die SQL quers. Es gibt 3 Verchieden
	all.sql 	= MYSQL Struktur + daten für die TestSeite
	strukt.sql 	= MYSQL Struktur fürs CMS
	inserts.sql	= MYSQL Daten für die TestSeite


Im Ordner Messages befdinen scih unterordner mit jedem mitarbeiter.
	Hier können Notizen für jeweils eine abgelget werden.
